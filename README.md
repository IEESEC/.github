# 💻 IEESEC | Software Engineering Community @ IEE.IHU.GR
Καλώς ήρθατε στο επίσημο GitHub Organization της **IEESEC** (IEE Software Engineering Community) του τμήματος [Μηχανικών Πληροφορικής και Ηλεκτρονικών Συστημάτων](https://iee.ihu.gr/) του ΔΙΠΑΕ!

## 🚀 Ποιοι είμαστε;
Είμαστε μια φοιτητική ομάδα με επίκεντρο τη **Μηχανική Λογισμικού (Software Engineering)**. Στόχος μας είναι να γεφυρώσουμε το κενό μεταξύ των ακαδημαϊκών γνώσεων και της πραγματικής αγοράς εργασίας μέσα από τη συνεργασία και το Open Source Development.

## 🛠 Τι κάνουμε;
- 🤝 **Collaborative Projects:** Αναπτύσσουμε εφαρμογές και εργαλεία σε ομάδες, μαθαίνοντας Git, Code Reviews και Agile στην πράξη.
- 🧠 **Knowledge Sharing:** Ανταλλάσσουμε πόρους, tutorials και βέλτιστες πρακτικές για διάφορα tech stacks.
- 🛠️ **Workshops & Meetups:** Διοργανώνουμε συναντήσεις και coding sessions.
- 💼 **Networking & Competitions:** Δικτυωνόμαστε μεταξύ μας και πάμε σε διαγωνισμούς/συνέδρια!

## 🌿 Πώς να συμμετέχεις;
Η κοινότητα είναι ανοιχτή σε όλους τους φοιτητές του τμήματος, ανεξαρτήτως έτους ή επιπέδου γνώσεων!

1.  👉 **Μπες στο Discord μας:** Η οργάνωση των projects γίνεται στο [Discord Server μας](https://discord.gg/2xHBsHMKy7)!
2.  🎯 **Pick a Project:** Δες τα Project μας και διάλεξε σε ποιο θα σου άρεσε να συμμετάσχεις!
3.  🧩 **Contribute:** Κάνε fork, δούλεψε στο δικό σου branch και στείλε ένα Pull Request!
